# Shadercuts (1.12.2 Forge)

Quick keybinds for controlling OptiFine shaders. Tested with OptiFine G5 and G6-pre1. 

| Action              | Default Key |
|---------------------|:-----------:|
| Toggle Shaders      |     `B`     |
| Open Shaders Menu   |     `N`     |
| Open Shader Options |     `M`     |
